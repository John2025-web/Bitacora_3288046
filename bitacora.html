Proyecto Final "1000 Conceptos"

1. Procesador

El procesador es el cerebro del computador, encargado de ejecutar instrucciones y coordinar las operaciones del sistema.

2. CPU

La Unidad Central de Procesamiento (CPU) es el componente principal de un ordenador que interpreta y ejecuta instrucciones.

3. Unidad de procesamiento gráfico

La GPU es un procesador especializado en cálculos gráficos y de procesamiento paralelo.

4. GPU dedicada

Tarjeta gráfica independiente que se instala en la computadora para ofrecer mayor potencia de procesamiento gráfico.

5. GPU integrada

Unidad de gráficos incorporada dentro del procesador o la placa base, de menor rendimiento que una GPU dedicada.

6. Núcleo (core)

Cada núcleo de un procesador puede ejecutar instrucciones de manera independiente, lo que permite multitarea.

7. Hilo (thread)

Secuencia de instrucciones que un procesador puede ejecutar; con la tecnología Hyper-Threading, un núcleo puede manejar varios hilos.

8. Caché L1

Memoria de acceso ultrarrápido ubicada dentro del procesador, muy pequeña pero extremadamente veloz.

9. Caché L2

Memoria caché intermedia entre L1 y L3, mayor en tamaño que L1 y más lenta, pero aún rápida.

10. Caché L3

Memoria caché compartida entre los núcleos de un procesador, más grande que L1 y L2 pero menos veloz.

11. Overclock

Práctica de aumentar la velocidad de reloj de un procesador para mejorar su rendimiento más allá de las especificaciones oficiales.

12. Zócalo de CPU

Ranura en la placa base donde se instala el procesador.

13. Placa base (motherboard)

Placa principal que conecta todos los componentes de la computadora, como CPU, RAM y dispositivos de almacenamiento.

14. BIOS

Sistema básico de entrada y salida que inicializa y prueba el hardware al encender el equipo.

15. UEFI

Interfaz de firmware moderna que reemplaza a la BIOS, con más funciones y mejor compatibilidad.

16. Firmware

Software de bajo nivel integrado en dispositivos electrónicos para controlarlos.

17. Chipset

Conjunto de circuitos en la placa base que gestionan la comunicación entre CPU, RAM, GPU y almacenamiento.

18. Memoria RAM

Memoria volátil que almacena datos temporales de los programas en ejecución.

19. DDR4

Tipo de memoria RAM de cuarta generación, rápida y eficiente en consumo de energía.

20. DDR5

Última generación de memoria RAM con mayor velocidad y ancho de banda que DDR4.

21. ECC (memoria)

Memoria con corrección de errores, utilizada en servidores para mayor fiabilidad.

22. Latencia CAS

Tiempo que tarda la memoria RAM en responder a una solicitud de acceso.

23. Swap

Espacio en disco utilizado como memoria virtual cuando la RAM se llena.

24. Memoria virtual

Técnica que utiliza parte del almacenamiento como extensión de la memoria RAM.

25. Unidad de estado sólido (SSD)

Dispositivo de almacenamiento rápido que utiliza chips de memoria flash en lugar de discos giratorios.

26. NVMe

Interfaz de alta velocidad para SSD que aprovecha el bus PCIe.

27. M.2

Formato físico de almacenamiento compacto, común en SSD modernos.

28. SATA

Interfaz estándar para conectar discos duros y SSD.

29. HDD (disco duro)

Dispositivo de almacenamiento tradicional que utiliza platos magnéticos giratorios.

30. RAID

Configuración de varios discos para mejorar rendimiento o redundancia.

31. Controladora RAID

Dispositivo que gestiona la organización y funcionamiento de discos en un arreglo RAID.

32. Hot swap

Capacidad de conectar o desconectar hardware sin apagar el sistema.

33. Fuente de poder (PSU)

Dispositivo que convierte la corriente eléctrica para alimentar los componentes del PC.

34. Modular (PSU)

Fuente de poder que permite conectar solo los cables necesarios.

35. Eficiencia 80 PLUS

Certificación que garantiza eficiencia energética en fuentes de poder.

36. Refrigeración líquida

Sistema avanzado de enfriamiento que utiliza líquido para disipar calor del procesador.

37. Ventilador PWM

Ventilador controlado por modulación de pulsos, que ajusta la velocidad según la temperatura.

38. Disipador térmico

Componente que transfiere el calor de un chip a un medio de enfriamiento.

39. Termal paste (pasta térmica)

Compuesto que mejora la conducción térmica entre CPU y disipador.

40. Sensor de temperatura

Dispositivo que mide la temperatura de los componentes electrónicos.

46. G-Sync

47. FreeSync
